## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript REST API boilerplate.

## Features
- [x] Database. Support [Prisma ORM](https://www.prisma.io/docs/orm/overview/introduction/what-is-prisma) and [PostgreSQL](https://www.postgresql.org/) 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
